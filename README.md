# NR Invisible Grills - Professional Website

A modern, responsive, and SEO-optimized website for NR Invisible Grills built with React, Vite, and CSS3.

## 🌟 Features

### 🎯 User Experience
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Smooth Navigation**: Seamless scrolling and navigation between sections
- **Mobile Menu**: Hamburger menu for mobile devices
- **Fast Loading**: Optimized with Vite for lightning-fast performance

### 🎨 Design
- **Modern UI**: Professional and clean design with gradient accents
- **Consistent Styling**: Global CSS variables for easy customization
- **Animations**: Smooth hover effects and transitions
- **Color Scheme**: Professional blue and orange theme

### 🔍 SEO Optimization
- **Meta Tags**: Comprehensive meta tags for better search engine visibility
- **Open Graph Tags**: Social media integration with OG tags
- **Semantic HTML**: Proper heading hierarchy and semantic elements
- **Performance**: Optimized for Core Web Vitals

### 📱 Sections Included
1. **Header** - Sticky navigation with logo and call-to-action button
2. **Hero** - Eye-catching banner with key value propositions
3. **Features** - 6 key benefits of invisible grills with icons
4. **Services** - 6 different service offerings
5. **About** - Company information and highlights
6. **Gallery** - Project showcase with overlay effects
7. **Testimonials** - Customer reviews and ratings
8. **Process** - Installation steps and workflow
9. **Comparison** - NR vs Traditional Grills comparison table
10. **CTA** - Contact form and contact information
11. **Footer** - Links, social media, and company information

## ⚡ Installation & Setup

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Quick Start

1. **Navigate to project directory**
   ```bash
   cd NetWebsites
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   The site will be available at `http://localhost:5173/`

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
