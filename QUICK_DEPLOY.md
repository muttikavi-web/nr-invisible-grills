# NR Invisible Grills - Quick Deployment to Vercel

## 📋 QUICK START - 5 Steps Only

### Step 1: Build Your Project
```bash
npm run build
```
✅ This creates a `dist/` folder ready for deployment

### Step 2: Create GitHub Account & Repo
- Sign up at https://github.com
- Create new repository called `nr-invisible-grills`

### Step 3: Push Code to GitHub
```bash
git init
git add .
git commit -m "Initial commit: NR Invisible Grills website"
git remote add origin https://github.com/YOUR_USERNAME/nr-invisible-grills.git
git branch -M main
git push -u origin main
```

### Step 4: Deploy to Vercel
- Go to https://vercel.com
- Sign up with GitHub
- Click "New Project"
- Select your `nr-invisible-grills` repository
- Click "Deploy"

### Step 5: Done! 🚀
Your site will be live at: `https://nr-invisible-grills.vercel.app`
